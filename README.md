# 👨‍💻 Ayush Sharma - Virtual Resume & Developer Portfolio

A modern, responsive, and animated personal portfolio built to showcase my technical projects, software engineering skills, and professional experience. 

Currently deployed live on **Vercel**: [Insert Your Vercel Link Here]

## ✨ Features
* **Dynamic Rendering:** Project cards and UI elements are generated dynamically via JavaScript, allowing for easy updates and future backend integration.
* **Scroll Animations:** Utilizes the Intersection Observer API for smooth, performance-friendly "fade-in-up" animations as the user scrolls.
* **Responsive Design:** Built with CSS Grid and Flexbox to ensure a flawless UI across mobile, tablet, and desktop displays.
* **Modern UI/UX:** Features a clean light-mode aesthetic, custom CSS variables for thematic consistency, and modern typography (Poppins).

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, Vanilla JavaScript (ES6+)
* **Styling Architecture:** Pure CSS with CSS Variables for easy theming (no external CSS frameworks, demonstrating core styling proficiency).
* **Deployment:** CI/CD pipeline integrated via GitHub to Vercel.
